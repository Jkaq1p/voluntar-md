# Voluntar.md - Platformă de voluntariat

Acest proiect reprezintă sarcina individuală pentru practica tehnologică (120 de ore). 
Este un site web complet, creat pentru a prezenta o platformă fictivă de voluntariat numită „Voluntar.md”.

## 🛠️ Tehnologii utilizate
* **HTML5:** Pentru structura semantică a celor 6 pagini.
* **CSS3 & Bootstrap 5:** Pentru design, flexbox, grid și layout responsive (adaptat pentru mobil, tabletă și desktop).
* **JavaScript:** Pentru interactivitate (validare formular, filtrare proiecte, galerie foto lightbox, contor animat etc.).
* **Fonturi & Iconițe:** Google Fonts (Poppins, Open Sans) și Font Awesome 6.

## 📁 Structura Site-ului
1. **Acasă (`index.html`)** - Prezentare generală, proiecte active, impact (counter animat).
2. **Proiecte (`projects.html`)** - Grila cu toate proiectele și sistem de filtrare (DOM & Locație).
3. **Cum ajuți (`help.html`)** - Modalități de implicare și secțiune FAQ (Accordion).
4. **Galerie (`gallery.html`)** - Galerie foto interactivă cu filtrare pe categorii și Lightbox.
5. **Blog (`blog.html`)** - Articole și noutăți.
6. **Contact (`contact.html`)** - Formular de înregistrare voluntar (cu validare JavaScript) și hartă.
